# Metodologia (skrót)

## Value
Value oznacza, że według modelu przewaga (edge) jest dodatnia dla danej linii/kursu.

## Edge
Edge to przewaga modelu nad rynkiem:

`edge = consensus_probability × best_price − 1`

Wartość dodatnia wskazuje potencjalny zakład o dodatniej wartości oczekiwanej.

## Confidence
Confidence to poziom zgodności (prawdopodobieństwo konsensusu) dla wygranej drużyny.

## Uwagi
- Metryki są orientacyjne i zależą od jakości danych historycznych.
- Wahania kursów mogą wpływać na wartość edge w czasie.
